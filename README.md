📂 File Organizer (Python GUI)

A simple and efficient File Organizer Desktop Application built using Python and Tkinter that automatically organizes files into folders based on their file type.

---

🚀 Features

- 📁 Automatically sorts files into categories:
  - Documents
  - Images
  - Music
  - Others
- 🖥️ Simple and user-friendly GUI (Tkinter)
- ⚡ One-click file organization
- 🧠 Clean and structured file management

---

🛠️ Tech Stack

- Python 🐍
- Tkinter (GUI)
- OS Module
- Shutil Module

⚙️ How It Works

1. Select a folder
2. Click on "Organize Files"
3. The program:
   - Reads all files
   - Identifies file types
   - Creates folders (if not exist)
   - Moves files into respective folders

---

▶️ How to Run

1. Install Python (if not installed)
2. Clone this repository:
   git clone https://github.com/your-username/file-organizer.git
3. Navigate to project folder:
   cd file-organizer
4. Run the script:
   python main.py
